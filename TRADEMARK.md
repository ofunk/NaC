# Marken- Und Namensregel

Die Open-Source-Lizenzen dieses Repositories gewähren Rechte am Code und an den
Dokumenten. Sie gewähren keine Markenrechte an `funktion8`, `NaC`,
`Notariat as Code`, Logos, Marktplatznamen oder sonstigen Kennzeichen.

## Erlaubt

- Nennung zur Attribution, etwa: "Based on NaC: Notariat as Code by
  funktion8 / ofunk".
- Beschreibung eines Forks als technisch auf NaC basierend.
- Verweise auf das Originalrepository unter
  `https://github.com/ofunk/NaC`.

## Nicht Erlaubt Ohne Zustimmung

- Ein Fork, Dienst oder Produkt darf nicht so auftreten, als sei er von
  funktion8 / ofunk freigegeben, zertifiziert oder betrieben.
- `funktion8 - NaC` darf nicht als eigener Marktplatz-, Produkt- oder
  Anbietername eines unabhängigen Forks verwendet werden.
- Logos oder sonstige Kennzeichen dürfen nicht für irreführende Herkunfts-,
  Support- oder Freigabeaussagen genutzt werden.

## Praktische Regel

Forks sollen ihren eigenen Namen tragen und sichtbar auf das Original verweisen.
So bleibt das Projekt offen nutzbar, ohne Herkunft und Verantwortung zu
verwischen.
