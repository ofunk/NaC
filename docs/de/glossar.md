# Glossar Für Nicht-Technische Leser

Dieses Glossar erklärt NaC-Begriffe so, wie sie für einen Notariatsbetrieb
relevant sind.

| Begriff | Bedeutung Für Das Büro |
| --- | --- |
| Git | Das versionierte Protokoll. Es zeigt, wer wann welche Regel, Vorlage oder Entscheidung geändert hat. |
| Repository | Der gemeinsame Arbeitsordner mit Regeln, Usecases, Workflows, Tests und Nachweisen. |
| CLI | Eine Kommandozeilen-Schnittstelle: ein eindeutig benannter Arbeitsauftrag an den Computer. In NaC ist sie die lokale, prüfbare Ausführungsschicht, die später auch von UI, Plugins oder CI gestartet werden kann. |
| Privater Fork | Die eigene, nicht öffentliche Kopie für ein konkretes Büro oder Pilotprojekt. Dort werden lokale Rollen und Betriebsregeln gepflegt. |
| Pull Request | Ein geprüfter Änderungsvorschlag. Er ersetzt ungeprüfte Direktbearbeitung an wichtigen Regeln. |
| Review | Die fachliche oder technische Prüfung, bevor eine Änderung verbindlich wird. |
| Merge nach `main` | Die Änderung wird in den verbindlichen Hauptstand übernommen. |
| Knowledge Graph / KG | Die strukturierte Liste offener Angaben, Dokumente, Entscheidungen, Gates und Nachweise für einen Vorgang. |
| Usecase | Eine konkrete notarielle Vorgangsart, zum Beispiel Immobilienkaufvertrag oder GmbH-Gründung. |
| Workflow | Ein wiederholbarer Ablauf aus Eingabe, Prüfung, Freigabe und Nachweis. |
| Plugin | Ein Begleitpaket für eine bestimmte technische oder fachliche Fähigkeit, zum Beispiel Kartenleser-Readiness oder Registerpfad. |
| Quality Gate | Ein automatischer Prüflauf. Er zeigt, ob Tests, Datenschutzregeln, Sprachregeln und KG-Regeln eingehalten werden. |
| SBOM / AI-SBOM | Eine Lieferkettenliste für Software, Modelle, externe Dienste und lokale Komponenten. |
| Evidence | Ein Nachweis oder eine Nachweisreferenz. Im öffentlichen Repo stehen nur Metadaten, keine echten Mandatsdokumente. |
| Readiness | Eine Bereitschaftsprüfung. Sie sagt, ob ein lokaler Arbeitsplatz oder Plugin-Pfad technisch vorbereitet ist. |
| Trockenlauf | Eine Planvorschau ohne produktive Schreibaktion und ohne echte Mandatsdaten. |

## Merksatz

NaC macht Regeln, offene Fragen und Freigaben sichtbar. Es ist nicht der Ort für
echte Akteninhalte.
